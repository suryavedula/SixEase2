The next generation of wealth advisory

A companion layer on top of the bank's existing CRM that lets one relationship manager give every client the hyper‑personalised, "feel‑known" care today reserved for a handful — at scale, with the human always in control.

“AI remembers. The RM connects.”

In a world flooded with AI, the scarce luxury is feeling known. The AI remembers everything — values, holdings, life events, promises — so the human can show up as if they remembered everything. AI suggests & explains; the RM recommends; the client always decides.

Core The product (MVP)

1 · UI

Generative, personalized UI

RM summons views on demand and sets a default entry view. Same data, many presentations.

2 · Engine

Personalized portfolio at scale

DNA‑driven, same‑sector, CIO‑constrained swaps with explainable fit scores across the whole book.

3 · Signals

News + CRM‑driven alerts

Watchlist = holdings ∪ DNA themes. Near‑real‑time, ranked, deduped queue.

4 · Voice

Tailored message + voice

Locked facts rendered in the client's style; query & dictate by voice.

5 · Notes

Voice → CRM notes

Dictation becomes a structured note (RM approves) + proposed DNA updates & tasks.

6 · Agentic

Tasks with safe autonomy

Agent auto‑runs research/analysis; outward & irreversible actions stay with the RM.

Rules Non‑negotiable principles

👤 Human‑in‑the‑loop

Every result ends in an action the RM takes. Nothing reaches the client automatically.

🔍 Traceability

Every alert/suggestion cites its evidence: the CRM note, the news item, the CIO view.

🛡 Strategy preserved

Personalisation is at the asset level only — the mandate & CIO weights never change.

🔗 CRM companion

We read the existing CRM and write drafts back. We never replace it.

Note The grounding contract

The LLM chooses which widget and arranges it and writes narrative — but every number (price, weight, fit, CIO rating) is produced by a data tool from SIX / portfolio / CRM. The model is a translator, not an author. This single rule is what makes the workbench trustworthy.
