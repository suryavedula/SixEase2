# SIX Web API

The SIX Web API grant you access to financial instrument prices and reference data.

The API authentication is based on certificates and private key that will be provided separately.

The documentation and samples for the Web API are available here: [SIX Web API Documentation](https://web.apiportal.six-group.com/portal/bfi/documentation)
You can test the API on the [API Portal](https://web.apiportal.six-group.com/portal/bfi/catalog)

---

**Important**, you have to use the pre-production enviroment: [https://api-preprod.np.six-group.com/web/v2](https://api-preprod.np.six-group.com/web/v2)
